# Physics Calculator

Will calculate all possible values from a given amount of properties.

## Usage

```javascript
 // populate with all given values
 var properties = {...}

```

run the program in node. Output will be a list of all calculated properties

## Key

```javascript

t // change in time

xI // initial x
xF // final x
xC // change in X

vI // initial velocity
vF // final velocity
vC // change in velocity

a // acceleration
g // gravity

```
